# sampleCorp

Sample web page frontend design VueJs which is a sample for little businesses.

tech stack|
---|
Vue|
Vite|
FontAwesome|

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
